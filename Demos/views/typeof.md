## typeof

- Number -> number  
- String -> string  
- Boolean -> boolean  
- undefined -> undefined  
- null -> object(不存在对象)  
- NaN -> number  
- Object -> function (构造函数)
- array -> object
- object -> object
- date -> object
- function -> function

**typeof 无法区别 date,array, null, object, 通过constructor/instanceOf 来判断**
